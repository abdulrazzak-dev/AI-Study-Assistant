About StudyAI

AI Study Assistant is a front-end project that uses the Google Gemini API to generate structured learning material from any topic you provide.

Features
  • AI-generated notes, summaries & quizzes
  • Key points & personalised study tips
  • Search history saved in Local Storage
  • Dark / Light mode with preference saved
  • Fully responsive & mobile-friendly
Tech Stack
HTML5 CSS3 Bootstrap 5 JavaScript ES6 jQuery 3 Gemini API Local Storage
Configure Gemini API Key

To use the AI Study Assistant, you need a Google Gemini API Key.

  1. Go to Google AI Studio to get a free key.
  2. Paste your key below. It will be saved securely in your browser's local storage.
